# ERC-8004 Research Findings

## Existing Tooling
- **@agentic-trust/8004-ext-sdk**: npm package for ENS resolution and identity lookups.
- **awesome-erc8004**: GitHub repo with curated resources (Identity, Reputation, Validation registries).
- **8004.qnt.sh**: Solana registry integration.
- **8004market.io**: Discovery indexer for agent collections.

## Opportunities for Agent Receipts
- Integrate identity resolution from `@agentic-trust/8004-ext-sdk` into the receipt cards.
- Map on-chain addresses to verified agent profiles from the Identity registry.
- Provide a "Proof of Agent" badge on receipts using ERC-8004 verification.
