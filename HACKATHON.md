## Project Concept: "Trusted Agent Commerce" (working title)

**The pitch:** An end-to-end demonstration of a trusted autonomous agent that:
1. Has verified onchain identity (ERC-8004 #22945)
2. Discovers services and other agents (pawr.link A2A)
3. Pays for services autonomously (x402 + Bankr)
4. Maintains reputation through verifiable receipts
5. Operates within safety guardrails (SOUL.md security rules)
6. Does all of this live, in production, during the hackathon

**What makes it real:** This isn't a demo. Clawlinker already does this. The hackathon submission is documenting and extending what's already running.
